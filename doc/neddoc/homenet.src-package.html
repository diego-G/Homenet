Package homenet.src

Name Type Description
Copy simple module

This module makes possible to choose if the incoming message is interesting for our statistics or its useless. Depending which kind of packet is, it is processed through throughput meter or it is sent to the network.
Decider simple module

This module decides if we are going to use the active or passive mode. We will use two variables, passive and emitter, to take the decision.
EthernetInterfaceDrops compound module

Ethernet network interface with Drops generator. Complements EtherMAC and EtherEncap with an output queue for QoS and RED support.
EthernetInterfaceThM compound module

Ethernet network interface with thorughput meter. Complements EtherMAC and EtherEncap with an output queue for QoS and RED support.
HomenetHost compound module

IP host with TCP and UDP layers and PING application.
HostWithDropsGenerator compound module

IP host with TCP and UDP layers and PING application. Besides, it has a drops generator in order to loose packets when neccesary.
Receiver compound module

This module receives the information and make the right calculations to obtain the end-to-end delay and jitter.
Stats compound module

This module is very important. Inside of it, we will take the measurements and record the results. There will be one in emitter and another in receiver mode.
ThM simple module

Throughput measurement utility module.
Trasmitter compound module

This module is the generator of traffic. It takes the time when the packet was created and the moment when it arrives to the module. This data is attached in the packet.
